Hola a  todos! 
Este Readme se creo en funcion de unificar la colaboracion entre todos los integrantes y asi evitar pisarnos durante el desarrollo del mismo.

Anteriormente teniamos un style.css para toda la web, esto imposibilitaba el correcto desarrollo ya que si alguien tenia modificaciones 
en la linea 32 y otra persona ya habia realizado cambios desde la linea 20 a la 40 hasta que no haya hecho el push 
no se sabia que linea era la correcta,esto iba a generar problemas a la hora de crear un merge a la master 
porque entrarian varios conflictos en el archivo.
Esto se resolvio separando un Style para cada HTML, 
el formato seria "style-" y despues del guion iria el nombre de la pagina a la que corresponde
Ejemplos: 
style-index.css para index.html
style-login.css para login.html
style-ficticio.css para ficticio.html

Lo que restaria hacer es limpiar los estilos que quedaron de las otras paginas html. 
Es decir borrar codigo que no se utilice en cada html anclado.
